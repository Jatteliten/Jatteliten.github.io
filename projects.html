Current project

Tattoo website with booking capabilities

I am currently working on a website that should allow a tattoo artist to show customers what hours are available to book.
The artist should also be able to book sessions and then later add information about the session once it has been completed.

  • Pages that the artist wants to serve to customers:
  • Home page
  • About me
  • Frequently asked questions
  • Booking calendar with available hours
  • Booking form
  • Free flash images
  • Portfolio from Instagram

  • Admin functionality that the artist wants:
  • Add bookable dates that are displayed in a calendar
  • Set a bookable date as standard, touch up or drop in.
  • Show a "latest news" section that can me altered by the admin
  • Be able to change what instagram post is being displayed as a portfolio
  • Book customers on available or unavailable dates
  • Upload images of available flash-sheets
  • Add information such as cost and notes to booked sessions.

The backend is written in Java with Spring Boot.
The Frontend is written with Thymeleaf.
Current database is Postgresql
Images are being stored in an AWS S3 bucket.
